# Frontend - Painel Pastoral

Aplicação web do projeto **Painel Pastoral**.

## Stack

- React 18
- TypeScript 5
- Vite 5
- Tailwind CSS v3
- Zustand com persistência
- TanStack Query
- React Hook Form + Zod
- Framer Motion
- Recharts

## Scripts

```bash
npm install
npm run dev
npm run build
```
